Framebuffer to png

What is fb2png:
A util to take screenshots from the framebuffer.

Usage:
fb2png /dev/fb0 outfile.png
or
cat /dev/fb0 > infile.raw
fb2png infile.raw outfile.png 0 800 600 16

Download:
fb2png version 0.1

Install:
make
make install
